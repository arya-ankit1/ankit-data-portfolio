# Ankit Data Portfolio

A curated portfolio of end-to-end analytics, machine learning, and data engineering projects designed to demonstrate business impact, technical depth, and production-style thinking for data roles in fintech, consulting, and product companies.

## Portfolio Projects

### 1. Credit Card Fraud Detection
**Focus:** Machine Learning, imbalanced classification, model evaluation, fraud analytics  
**Stack:** Python, Scikit-learn, SQL, Tableau  
**Impact:** Built a fraud detection workflow on 280K+ transactions with strong precision-recall tradeoff and business-oriented thresholding.

### 2. Fintech Customer Churn Prediction
**Focus:** Retention analytics, customer segmentation, predictive modelling  
**Stack:** Python, XGBoost / Scikit-learn, SQL, Power BI  
**Impact:** Identified high-risk customer cohorts and translated model outputs into retention actions.

### 3. Azure Data Engineering Pipeline
**Focus:** Cloud data engineering, ETL/ELT, orchestration, reporting  
**Stack:** Azure Data Factory, Azure Data Lake, Azure SQL, Python, Power BI  
**Impact:** Designed a scalable pipeline integrating multiple sources to support near real-time analytics.

### 4. Sales & Profitability Intelligence Dashboard
**Focus:** Business intelligence, KPI reporting, commercial analytics  
**Stack:** SQL, Tableau, Excel, Python  
**Impact:** Built an executive dashboard highlighting revenue trends, profitability gaps, and regional opportunities.

### 5. UK Immigration Forecasting
**Focus:** Time-series forecasting, policy analytics, storytelling  
**Stack:** Python, Prophet / ARIMA, Tableau, Excel  
**Impact:** Forecasted migration trends using multi-year public data and converted analysis into decision-ready insights.

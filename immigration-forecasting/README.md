# UK Immigration Forecasting and Policy Analytics

## Project Overview
This project extends a dissertation-style analysis into a portfolio-grade forecasting solution focused on migration trends, policy interpretation, and data storytelling.

## Business / Policy Problem
Public policy and planning decisions rely on understanding migration trends over time. The objective is to identify historical patterns and forecast future movement using time-series techniques.

## Dataset
- Multi-year UK immigration and migration trend data
- 10+ years of historical observations
- Government / public policy data sources

## Tech Stack
- Python
- Pandas
- Statsmodels / Prophet
- Tableau
- Excel

## Workflow
1. Historical trend analysis  
2. Data cleaning and time-series preparation  
3. Seasonality and trend decomposition  
4. Forecast modelling using ARIMA / Prophet  
5. Scenario analysis and policy storytelling  
6. Dashboarding and visualization  

## Key Analysis
- Reviewed long-term migration trend movement
- Evaluated changes across years and policy periods
- Compared baseline and forecast scenarios
- Converted model outputs into clear visual narratives

## Results
- Built a migration forecasting solution using multi-year historical data
- Produced scenario-based forecasting outputs
- Created dashboards and charts suitable for non-technical stakeholders
- Strengthened the project with applied analytics and communication framing

## Business / Policy Impact
This project demonstrates:
- Strong time-series forecasting capability
- Ability to work with real-world public datasets
- Translation of technical analysis into policy-oriented insights
- Executive-style communication of uncertain future trends

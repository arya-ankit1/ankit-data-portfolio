# Sales & Profitability Intelligence Dashboard

## Project Overview
This project builds a business intelligence solution to analyze revenue, profitability, product performance, and regional sales trends.

## Business Problem
Commercial teams often need a single view of performance across products, regions, and customer segments. The goal is to create a dashboard that supports faster decisions around growth and margin optimization.

## Dataset
- Retail / consumer sales dataset
- 100K+ transaction records
- Sales, orders, products, customers, and region-level dimensions

## Tech Stack
- SQL
- Tableau
- Excel
- Python

## Workflow
1. Data extraction and cleaning  
2. SQL metric layer creation  
3. KPI definition for revenue, margin, and growth  
4. Dashboard design in Tableau  
5. Trend, regional, and product performance analysis  

## Key Analysis
- Measured revenue, profit, and margin by region and category
- Identified top and bottom performing products
- Evaluated monthly growth trends
- Highlighted regional opportunities and profitability gaps

## Results
- Built an interactive dashboard on 100K+ sales records
- Standardized business KPI reporting
- Surfaced margin improvement opportunities
- Improved stakeholder visibility into performance trends

## Business Impact
This dashboard enables:
- Faster review of commercial performance
- Improved tracking of profitability
- Better prioritization of underperforming regions/products
- Clearer communication with business stakeholders

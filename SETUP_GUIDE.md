# Setup Guide

## How to use this portfolio
1. Create a separate GitHub repository for each project folder, or keep them in one monorepo.
2. Upload the relevant README, notebook, SQL, and dashboard screenshots.
3. Replace placeholder notes with actual analysis notebooks and visuals.
4. Pin your top 5 repositories on GitHub.
5. Add the GitHub link to your CV and LinkedIn Featured section.

## Recommended repo descriptions
- fraud-detection-ml: End-to-end fraud detection pipeline using Python, SQL, and BI dashboards
- fintech-churn-prediction: Customer churn modelling and retention analytics for fintech use cases
- azure-data-pipeline: Azure data engineering pipeline for scalable reporting and analytics
- sales-analytics-dashboard: Tableau sales and profitability dashboard with KPI intelligence
- immigration-forecasting: Time-series forecasting and policy analytics on UK migration data
